# Компоненты системы модерации

---

## BanRequestView
!!! info ""
    Информация о будущей блокировке:

    - `Пользователь` - пользователь, форма на бан которого отправлена (Упоминание и id)
    - `Модератор` - модератор, отправивший форму на бан (Упоминание)
    - `Причина` - причина выдачи бана
    - `Длительность` - пример - 30d
    - `Удалить сообщения за последнее время` - пример - 5d (опционально)
    - `Вложения` - доказательства нарушения, только при отправке формы через пкм по юзеру, не через `/voteban `(опционально)

    Голосование завершается при `4` голосах от `любых` пользователей, или при 1 голосе от пользователя с доступом `BAN_ACCESS`

    ### Необходимые права бота
    - `send_messages` - доступ к отправке сообщений в канал для логирования модерации (в случае если он настроен) 
    обязательно для отправки лога о выдачи бана (в случае положительного завершения голосования)
    - `ban_members` - для выдачи блокировки пользователю (в случае положительного завершения голосования)

!!! warning "Возможные ошибки"
    - Вы уже участвуете в голосовании
