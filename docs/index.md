# Nightcore

Добро пожаловать в официальную документацию **Nightcore** — многофункционального Discord бота для модерации, экономики и развлечений.

!!! tip "Официальный сервер поддержки"
    Присоединяйтесь к нашему [Discord-серверу](https://discord.gg/nDmYphcHk8), чтобы задать вопросы, пообщаться с другими пользователями и первыми узнавать об обновлениях.

!!! note "Документация"
    Полная документация по всем командам и системам доступна в соответствующих разделах меню слева.
    Если вы не нашли нужной информации, воспользуйтесь поиском (клавиша S).

!!! warning "Баги и недоработки"
    При нахождении таковых, используйте команду /bug_report для оповещения разработчиков.
