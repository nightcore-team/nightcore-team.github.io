# Компоненты системы кланов

---

## Клановый магазин
!!! info ""
    Информация о покупке:

    - Пользователь (упоминание и id)
    - Клан (название и id роли)
    - Баланс клана до покупки
    - Баланс клана после покупки
    - Название товара
    - Цена товара
    - Идентификатор покупки - id созданной ветки

    ### Доступ:
    CLANS_ACCESS

    ### Необходимые права бота:
    - send_messages - доступ к отправке сообщений в канал для логирования клановой системы (в случае если он настроен), обязательно для отправки лога одобрения или отклонения покупки
    - manage_threads - для закрытия и архивирования ветки
