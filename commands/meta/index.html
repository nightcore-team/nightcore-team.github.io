    Команды

    Общие команды, не попавшие в одну из систем

    /banner

    Отображает баннер пользователя

    Параметры

    • user - пользователь, чей баннер вы хотите получить (Упоминание или id)
    • ephemeral - видимость ответа бота True (Видно всем) или False (Видно только тому кто ввел команду) (опционален)

    Возможные ошибки

    • У пользователя нет баннера

    /avatar

    Отображает глобальный аватар пользователя

    Параметры

    • user - пользователь, чей аватар вы хотите получить (Упоминание или id)
    • ephemeral - видимость ответа бота True (Видно всем) или False (Видно только тому кто ввел команду) (опционален)

    /ping

    Получить текущую задержку бота

    /about

    Отображает общую информацию о боте:

    • ID бота
    • Текущую задержку
    • Дату создания
    • Количество серверов
    • Количество пользователей - количество записей в бд

    А так же техническую информацию:

    • Потребление памяти
    • Библиотеку и ее версию
    • Время работы - время работы с момента последнего рестарта

    Параметры

    • ephemeral - видимость ответа бота True (Видно всем) или False (Видно только тому кто ввел команду) (опционален)

    /join

    Бот заходит в определенный голосовой канал

    Доступ

    DEVELOPER

    Параметры

    • channel - канал, в который боту нужно зайти, голосовой

    /say

    Написать сообщение от лица бота в текущий канал (В котором была введена команда)

    Доступ

    DEVELOPER

    Параметры

    • text - сообщение, которое бот отправит
    • image - изображение, которое бот отправит

    /rolemembers

    Получить информацию об участниках с определенной ролью
    Отправляется сообщение с упоминаниями пользователей гильдии, у которых есть введенная роль

    Параметры

    • role - роль, участники которой будет показаны (Упоминание или id)
    • ephemeral - видимость ответа бота True (Видно всем) или False (Видно только тому кто ввел команду) (опционален)

    /role_selector

    Отправить сообщение c селект меню ролей для выдачи
    После ввода команды появляется модальное окно с селектором выбора нужных для выдачи ролей

    Ограничения для выбранных ролей:

    • Позиция роли не должна быть выше чем высшая роль бота
    • Роль не должна иметь права администратора
    • Позиция роли должна быть меньше чем высшая роль того кто ввел команду
    • Выбрать можно максимум 24 роли

    Доступ

    HEAD_MODERATION_ACCESS

    Необходимые права бота

    • send_messages - доступ к отправке сообщений в текущий канал для отправки сообщения с селектором

    Параметры

    • title - заголовок для сообщения с селектором
    • description - описание для сообщения с селектором
    • image_url - изображение для сообщения с селектором
    • цвет - цвет сообщения с селектором

    /bug_report

    Команда для оповещения разработчиков о баге
    После ввода вы увидите модальное окно с параметрами:

    • Полное описание проблемы - параметр для описания найденного бага, ограничение 300 символов
    • Прикрепите скриншот с ошибкой - параметр для вставки изображения (опционален)

    Сообщение будет отправлено в канал на нашем сервере поддержки

    /action

    Возможные действия:

    • Обнять - парное
    • Ударить - парное
    • Укусить - парное
    • Спать
    • Плакать
    • Радоваться
    • Грустить
    • Курить
    • Поцеловать - парное

    Параметры

    • action - действие
    • user - пользователь, с которым вы хотите совершить действие (В случае, если оно парное, иначе ввод не обязателен) (Упоминание или id)

    Возможные ошибки

    • Вы хотите использовать парное действие на себе
