# Общее про систему приватных комнат

---

!!! info "Использование"
    Когда вы заходите в настроенный голосовой канал, бот перекидывает вас в созданный канал (или в текущий, если ваша приватная комната уже существует) и выдает права на изменение канала и управление участниками в этом канале.

    При выходе создателя из своей из комнаты она удаляется

!!! info "Настройка"
    Для настройки системы нужен только один параметр - `create_private_channel`, который настраивается [здесь](../commands/config.md#config-private_channels)  
    Логирование настраивается [здесь](../commands/config.md#config-logging), параметр `private_rooms`
